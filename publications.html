Enhancing The User Experience of Interior Design Mobile Augmented Reality Applications

Authors: Ayman Kandil, Prof. Iyad Abu Doush, Bader Al-Juma
Paper

Abstract:
Intuitive user interface design is of utmost importance to mobile applications, especially when dealing with newer technologies like Augmented Reality (AR). With AR there is room for more innovation in the smartphone's user experience as the technology is just beginning to take off with more devices offering AR support. We find new ways to simplify and stream-line AR applications’ user experience by creating a prototype 3D furniture arrangement application and improving it through users’ feedback. The results show that small changes to the user interface and applying conventional mobile design principles found in other AR and non-AR applications improves the user experience. Our approach is based on simplifying the design by applying the tried and true concept of “less is more”.
