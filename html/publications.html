Enhancing The User Experience of Interior Design Mobile Augmented Reality Applications

Authors: Ayman Kandil, Prof. Iyad Abu Doush, Bader Al-Juma
Paper

Abstract:
Intuitive user interface design is of utmost importance to mobile applications, especially when dealing with new technologies like Augmented Reality (AR). In this paper, a user study for evaluating AR 3D furniture arrangement mobile application user experience is presented. In our gesture design, we used one hand to ease the use of the application. Firstly, the user interface is developed based on the literature recommendations and users evaluate it using a set of five tasks in terms of System Usability Scale (SUS), Handheld Augmented Reality Usability Scale (HARUS), task completion time, and the number of user errors. The obtained evaluation results are then used to alter the user interface. The research outcome can be used to help in developing a better user experience for a wider range of AR applications.
